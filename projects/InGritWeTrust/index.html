§ Project Description

Bastian Gasser, a communication designer from Tirol, Austria, reached out to me to create a short animation of an American coin, which typically bears the inscription "In God We Trust" For the launch of their mobile app, he wanted the text modified to "In Grit We Trust"—the app's name. Since the animation was intended for both mobile and desktop, I created renders in both 16:9 and 9:16 formats.

This was a small project, but sometimes, these little projects are the ones that inspire you the most. In this case, for the first time, I experimented with rendering multiple breakdown passes for this portfolio page. I found this to be the most time-consuming part of the entire project, so I’ve added it to my to-do list to create a tool that automates this process.
