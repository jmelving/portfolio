Â§ Project Description

I took part in the Alternate Realities competition by Pwnisher, which ended on June 1st 2021. This documents my one-month journey to completing my submission. My goal was to create a visually interesting and creative piece that utilized the competition template in an innovative way. For example, instead of simply using a giant moon in the background, I transformed it into a giant wheel that turns continuously. This element was meant to be symbolic, representative of the era of the Black Plague.

The first thing that I do for these kind of projects is drawing a little concept art for it so I have Reference for all the Assets I need to gather and what the composition could look like. Ive had about 5 Ideas which I first scribbled out and then painted some more detail into it.
